# Mindfulness i vardagen

Fem snabba praktikexempel du kan använda varje dag för att odla närvaro.

- Andningsövningar när du stannar upp
- Kort kroppsskanning
- Mindful målning av vardagliga uppgifter
- 1-minuters övningar
- Journalanteckningar
